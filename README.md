# 基础用法

```html
vue init webpack-simple vue-skeleton-screen
然后就有了这个目录
├── package.json
├── src
│   ├── App.vue
│   ├── assets
│   └── main.js
├── index.html
└── webpack.conf.js
npm 大法好 一顿install就行 不行就cnpm  再不行就yarn 
我在src下面随便写个vue文件
然后再写一个vue文件的js（随缘命名）
然后再跟目录下面整个webpack.skeleton.conf.js用来打包一个dist文件
然后再写一个skeleton.js (也就是即将插入index.html的东西)
然后再npm run lijiahang
然后再npm run dev 网速慢一点就能看见了


总结
npm install 
然后 npm run lijiahang

然后把network 网速弄成3g
```
